# escaladefr
escalade-france.fr est un projet de site internet spécialisé dans le rencensement d'un maximum de spot d'escalade en milieu naturel, ainsi qu'un recensement de toutes les salles d'escalade de France. Un coin boutique fonctionnant via des liens d'affiliation sera / est également en place afin d'aider à équiper les amateurs d'escalade.
